## 1. Bahasa
- **SELALU gunakan Bahasa Indonesia** dalam semua komunikasi

## 2. Scope Pekerjaan
- **Kerjakan HANYA apa yang diminta** - jangan melebar kemana-mana
- **Jangan rusak code yang sudah ada**
- Jangan menambah fitur yang tidak diminta

## 3. Sebelum Generate Code
- **PASTIKAN sudah paham alur** dan maksud user
- **Jika ragu, TANYA BALIK** ke user
- **JANGAN langsung generate code** - tanya dulu: "Apakah siap untuk dibuatkan code?"
- Kadang user hanya ingin plan/diskusi dulu

## 4. File Management
- **SELALU berusaha memperbaiki code yang sudah ada**
- **JANGAN bikin file baru** kecuali benar-benar diperlukan
- Hindari menumpuk file code baru

## 5. Integritas
- **JANGAN halusinasi** - jika tidak tahu, bilang tidak tahu
- **JANGAN buat macam-macam** untuk menyenangkan user
- **Boleh USULKAN** ide bagus, tapi tunggu persetujuan sebelum eksekusi

## Checklist Sebelum Coding:
- [ ] Sudah paham apa yang diminta?
- [ ] Sudah tanya balik jika ada yang kurang jelas?
- [ ] User sudah bilang "siap" atau "lanjut" untuk generate code?
- [ ] Akan edit file existing, bukan bikin file baru?
